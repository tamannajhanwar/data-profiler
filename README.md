# 📊 Data Analysis Project

## 📌 Project Overview

This project focuses on performing data analysis using Python. It includes data acquisition, data cleaning, exploratory data analysis (EDA), and visualization to extract meaningful insights from the dataset.

---

## 🎯 Objectives

* Load data from different sources (CSV, JSON, SQL, API)
* Clean and preprocess the data
* Handle missing values and inconsistencies
* Perform exploratory data analysis
* Visualize relationships between variables

---

## 🗂️ Dataset Description

The dataset contains customer-related information such as:

* Customer ID
* Age
* Gender
* Income
* Purchases
* Churn (whether the customer stopped purchasing)

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQLite / MySQL

---

## 🔍 Steps Performed

### 1. Data Acquisition

* Loaded data from CSV and JSON files
* Connected to SQL database
* Fetched data using APIs

### 2. Data Cleaning

* Handled missing values using mean/mode
* Corrected inconsistent data types
* Removed duplicate records
* Dropped irrelevant columns

### 3. Exploratory Data Analysis (EDA)

* Performed univariate analysis
* Conducted bivariate analysis (relationships between variables)
* Performed multivariate analysis using correlation and pair plots

### 4. Data Visualization

* Created histograms for distributions
* Generated bar plots for comparisons
* Used heatmaps for correlation analysis

### 5. Data Quality Checks

* Identified missing values
* Detected duplicates
* Checked for inconsistencies

---

## 📈 Key Insights

* Customer purchase behavior varies by gender
* Income has an impact on churn
* Missing values and inconsistent data required preprocessing


## 📝 Conclusion

This project demonstrates the complete data analysis pipeline from data collection to visualization. It highlights the importance of data cleaning and analysis in deriving meaningful insights.
